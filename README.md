# Weather Calculator Website

## Project Description

This is a simple Weather Calculator Website that allows users to check the current weather of any city. The application fetches real-time weather data from a weather API and displays useful information such as temperature, humidity, wind speed, and weather conditions.

The project is designed with a clean and responsive user interface, making it easy for users to quickly get weather updates.

## Features

* Search weather by city name
* Displays current temperature
* Shows humidity and wind speed
* Displays weather conditions (cloudy, sunny, rain, etc.)
* Simple and responsive design
* Real-time weather data using API

## Technologies Used

* HTML
* CSS
* JavaScript
* Weather API

## How to Use

1. Enter the city name in the search box.
2. Click the search button.
3. The website will display the current weather information for that city.

## Purpose of the Project

This project was created to practice JavaScript, API integration, and DOM manipulation while building a useful real-world web application.
